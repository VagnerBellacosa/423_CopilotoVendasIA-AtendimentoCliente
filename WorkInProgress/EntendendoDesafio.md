# Entendendo o Desafio

Agora é a sua hora de praticar e criar um projeto para o seu portfólio!

Neste desafio, você vai desenvolver um **Copiloto De Vendas Com IA Para Atendimento Ao Cliente**. A ideia é criar uma solução simples que ajude uma pessoa vendedora, atendente ou equipe comercial a entender melhor o cliente, organizar informações, sugerir boas respostas e apoiar o processo de venda.

Nas aulas, o Felipão usa uma **Loja Gamer** como exemplo. Porém, você pode adaptar o projeto para qualquer área, como moda, educação, beleza, tecnologia, imóveis, alimentação, serviços, produtos digitais ou qualquer outro contexto de vendas.

## O Que Criar

Você deve criar uma solução de IA voltada a vendas e atendimento ao cliente. Ela pode ser um chatbot, um copiloto, um agente, um simulador de atendimento, uma análise de oportunidades ou uma proposta simples de automação comercial.

Sua solução pode ajudar em tarefas como:

- Entender o que o cliente procura;
- Sugerir produtos, serviços ou abordagens comerciais;
- Responder dúvidas comuns;
- Ajudar a lidar com objeções;
- Sugerir uma próxima mensagem para o cliente;
- Organizar informações sobre o negócio;
- Apoiar a priorização de clientes, leads ou pontos de venda.

O objetivo não é substituir a pessoa vendedora, mas apoiar o atendimento com respostas mais claras, úteis e contextualizadas.

## Possíveis Caminhos Para O Projeto

Você pode escolher um recorte simples e desenvolver sua solução a partir dele.

- **Atendimento:** criar um chatbot que responda dúvidas comuns de clientes;
- **Vendas:** criar um copiloto que sugira argumentos comerciais e próximas mensagens;
- **Objeções:** criar uma IA que ajude a responder dúvidas, resistências ou inseguranças do cliente;
- **Priorização:** criar uma lógica simples para indicar quais clientes, leads ou pontos de venda têm mais potencial;
- **Conhecimento De Marca:** criar uma IA que use informações de uma live, documento ou base de conhecimento para responder com mais contexto;
- **Simulação:** criar exemplos de conversas entre cliente e vendedor para testar diferentes abordagens.

## Como Usar Uma Base De Conhecimento

Para melhorar as respostas da IA, você pode criar uma pequena base de conhecimento com informações do negócio escolhido.

```
knowledge/
  produtos.md
  perguntas-frequentes.md
  objecoes.md
  contexto-do-negocio.md
```

Se você seguir a alternativa da FYS, pode usar a transcrição da live como uma das principais fontes de contexto. Ela pode ajudar a IA a entender melhor a marca, os desafios apresentados e as oportunidades de solução.

## O Que Entregar

Crie um repositório no GitHub com os principais arquivos do seu projeto.

No `README.md`, explique:

- Qual tema você escolheu;
- Quem é o usuário principal da solução;
- Qual problema de vendas ou atendimento ela resolve;
- Qual abordagem você usou: chatbot, copiloto, agente, simulador, análise de dados ou outra;
- Quais informações foram usadas como base de conhecimento;
- Um exemplo de conversa, resposta ou análise gerada;
- Possíveis melhorias futuras.

Você não precisa criar uma aplicação completa. O mais importante é demonstrar que entendeu o problema, organizou bem a solução e usou IA de forma útil.

## Resultado Esperado

Ao final, você terá um projeto simples e prático mostrando como a IA pode apoiar vendas e atendimento ao cliente.

Use a loja gamer como inspiração inicial, explore a alternativa da FYS se quiser trabalhar com um contexto real, ou personalize o desafio com o tema que fizer mais sentido para você.
