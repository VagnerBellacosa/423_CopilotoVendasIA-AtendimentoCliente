> "Por trás de um PIX realizado em segundos, de uma compra aprovada no cartão e de um saque realizado em um caixa eletrônico, existe uma grande chance de um Mainframe estar trabalhando silenciosamente nos bastidores."

# IBM Mainframe

## O Gigante Invisível da Tecnologia

O Mainframe é um computador corporativo projetado para processar milhões de transações com extrema confiabilidade.

Ele é utilizado por:

- Bancos
- Seguradoras
- Governos
- Operadoras de Telecom
- Companhias Aéreas
- Grandes Varejistas

### Curiosidade

Quando você faz um PIX, consulta seu saldo ou utiliza um cartão de crédito, existe uma enorme probabilidade de que uma aplicação Mainframe esteja envolvida na operação.

### Principais Características

- Alta disponibilidade
- Segurança avançada
- Escalabilidade
- Processamento massivo
- Confiabilidade extrema

------

# z/OS

## O Sistema Operacional dos Mainframes

O z/OS é o principal sistema operacional da plataforma IBM Z.

Ele coordena:

- Memória
- Processadores
- Arquivos
- Segurança
- Redes
- Aplicações

### Pense Assim

Se o Mainframe é uma cidade inteira, o z/OS é o prefeito que coordena tudo para que a cidade continue funcionando sem parar.

------

# COBOL

## A Linguagem que Move o Mundo

COBOL significa:

Common Business Oriented Language

Criada em 1959, continua sendo uma das linguagens mais utilizadas para sistemas corporativos.

### Onde é utilizado?

- Bancos
- Previdência
- Folha de pagamento
- Seguros
- Sistemas financeiros

### Exemplo de uso

- Consulta de saldo
- Emissão de boleto
- Cálculo de impostos
- Processamento de benefícios

### Curiosidade

Milhões de linhas de COBOL continuam em produção no mundo inteiro.

------

# JCL

## O Maestro dos Processamentos Batch

JCL significa:

Job Control Language

É a linguagem utilizada para controlar a execução de programas no z/OS.

### O que faz?

- Executa programas
- Copia arquivos
- Organiza processos
- Automatiza rotinas

### Analogia

Imagine um diretor de cinema.

O COBOL é o ator.

O JCL é o diretor dizendo:

"Agora execute este programa."

------

# CICS

## O Rei das Transações Online

CICS significa:

Customer Information Control System

É responsável por processar milhares de transações simultâneas.

### Exemplos

- Consulta de saldo
- Saque em ATM
- PIX
- Aprovação de cartão

### Curiosidade

Alguns ambientes processam milhares de transações por segundo usando CICS.

# RACF

## O Porteiro do Mainframe

RACF significa:

Resource Access Control Facility

É o principal sistema de segurança da plataforma IBM Z.

### Responsabilidades

- Controle de usuários
- Senhas
- Perfis
- Permissões
- Auditoria

### Analogia

Se o Mainframe fosse um prédio, o RACF seria a portaria inteligente que decide quem pode entrar e para onde pode ir.
